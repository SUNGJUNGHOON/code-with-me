### 미션 2
for 문을 사용하여 2단을 출력하라.
```
2 X 1 = 2
2 X 2 = 4
2 X 3 = 6
2 X 4 = 8
2 X 5 = 10
2 X 6 = 12
2 X 7 = 14
2 X 8 = 16
2 X 9 = 18
```

### 미션 3
이중 for 문을 사용하여 단수를 입력 받을 수 있는 구구단을 출력하라.
```
단수를 입력하세요.
3

3 X 1 = 3
3 X 2 = 6
3 X 3 = 9
3 X 4 = 12
3 X 5 = 15
3 X 6 = 18
3 X 7 = 21
3 X 8 = 24
3 X 9 = 27

끝내시겠습니까? (예: y, 아니요: n)
y
```
y 입력 시 종료, n 입력 시 '단수를 입력하세요.' 메시지 노출 후 다시 입력
